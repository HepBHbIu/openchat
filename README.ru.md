Анонимный публичный чат — без регистрации, без истории, просто вводишь ник и общаешься.

## Возможности

| Возможность | Описание |
|-------------|----------|
| **Без регистрации** | Вводишь ник — общаешься |
| **Каналы** | Темы: IT, Пиво, Музыка, Игры и др. |
| **Анонимность** | Нет аккаунтов, нет логов, нет трекинга |
| **Децентрализация** | Любой может запустить сервер |
| **Реалтайм** | WebSocket, мгновенные сообщения |

## Быстрый старт

```bash
git clone https://github.com/HepBHbIu/openchat.git
cd openchat
npm install
npm start
```

Открой http://localhost:3001

## Как это работает

1. Открываешь страницу
2. Вводишь любой ник
3. Выбираешь канал
4. Общаешься!
