Anonymous public chat — no registration, no history, just enter a nickname and chat.

## Features

| Feature | Description |
|---------|-------------|
| **No Registration** | Pick a nickname, start chatting |
| **Channels** | Topics: IT, Beer, Music, Gaming, etc. |
| **Anonymous** | No accounts, no logs, no tracking |
| **Decentralized** | Anyone can run a server |
| **Real-time** | WebSocket, instant messages |

## Quick Start

```bash
git clone https://github.com/HepBHbIu/openchat.git
cd openchat
npm install
npm start
```

Open http://localhost:3001

## How It Works

1. Open the page
2. Enter any nickname
3. Pick a channel
4. Chat!

## API

| Endpoint | Description |
|----------|-------------|
| GET `/api/channels` | List channels |
| GET `/api/stats` | Server stats |
| POST `/api/channels` | Create channel |
